Remarks

The ObjectPropertiesUpdated event occurs when the simulator sends an ObjectPropertiesPacket containing additional details for a Primitive or Foliage data that is currently being tracked in the ObjectsPrimitives dictionary

The ObjectPropertiesUpdated event is also raised when a SelectObject(Simulator, UInt32) request is made and OBJECT_TRACKING is enabled
